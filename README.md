Wanky Web

Wanky Web is the official web presence for Wanky, a software-focused brand building modern, scalable digital products.

This repository serves as the foundation for Wanky’s public-facing website—covering brand identity, offerings, and future product touchpoints.

 Purpose

The goal of wanky-web is to:

Establish brand credibility and online presence

Clearly communicate what Wanky builds and stands for

Act as a launchpad for future products, case studies, and announcements

Scale cleanly as the company evolves (products, services, hiring, content)

This is not a throwaway marketing site.
It’s infrastructure for long-term positioning.

Tech Stack

Framework: Next.js (App Router)

Language: TypeScript

Styling: Tailwind CSS

UI Philosophy: Minimal, modern, performance-first

Deployment Target: Vercel
